# Website for the Truth Effect Database (TED)
This repository contains the overview website of the Truth Effect Database (TED). To learn more about how to use TED, [click here](https://slesche.github.io/ted-site/).

The database itself is stored on its own GitHub repository [here](https://github.com/slesche/truth-effect-database).

Information on the R-package to work with TED can be found [here](https://github.com/slesche/acdc-query).
